# PRODIGY_FS_03
📋 Employee Management System

The 👨‍💼 Employee Management System is a web-based application designed to simplify employee record management for organizations. It allows users to ➕ add, 🔍 search, 📋 view, 🗑️ delete, and manage employee information through an easy-to-use interface. The system stores employee details such as 👤 Name, 📧 Email, 🏢 Department, 💼 Designation, and 💰 Salary. Data is saved using 💾 Local Storage, ensuring records remain available even after refreshing the browser. This project helps improve efficiency, reduce paperwork, and organize employee data in a secure and user-friendly manner.

✨ Features
👤 Add Employee
📋 View Employee Records
🔍 Search Employees
🗑️ Delete Employee
💾 Local Storage Support
📱 Responsive Design
🎨 Clean & Modern User Interface
🛠️ Technologies Used

🎨 CSS3
⚡ JavaScript
💾 Browser Local Storage
